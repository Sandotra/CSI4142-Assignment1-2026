## Project Overview
This repository contains the Jupyter Notebook and datasets for Assignment 1. The goal of this project is to perform comprehensive Exploratory Data Analysis (EDA) on two distinct datasets from different domains.

## Datasets
Per the assignment requirements, the datasets are hosted directly in this repository to ensure the code runs without requiring local downloads.

### Dataset 1: Mobile Device Usage and User Behavior
* **Domain:** Technology/Mobile Behavior
* **Source:** Kaggle
* **Description:** Analysis of user behavior including app usage time, battery drain, and device models.

### Dataset 2: Used Car Sales (Audi, BMW, Ford, VW, etc.)
* **Domain:** Automotive
* **Source:** Kaggle
* **Description:** Analysis of resale values, mileage, transmission types, and fuel efficiency across various manufacturers.

## Technologies Used
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Libraries:** Pandas, Matplotlib, Seaborn
